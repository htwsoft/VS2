

Um diese .class Dateien / Klassen nutzen zu können:

    - Erstellt in eurem Verzeichnis, in dem ihr eure .java files habt einen Ordner namens "storageSystem" (Schreibweise wichtig!)
    - Kopiert euch und fügt die .class Datei in diesen Ordner ein, welche ihr nutzen möchtet
    - In der .java Datei, in der ihr die Storage Klasse nutzen wollt müsst ihr folgendes einfügen:
        -> oben bei den import Zeilen
        -> zB. für die StorageMessage Klasse: "import storageSystem.StorageMessages;"
            => bedeutet:                                    ^             ^
                                           der Package-(Ordner)name    die .class die ihr nutzen möchtet
                                           (NICHT änderbar, wenn man
                                           den Ordner umbenennt!
                                           Ist als package in den
                                           .class Dateien so festgelegt!)
    - Beim Befehl zum Kompilieren eurer Datei müsst ihr nichts weiter angeben! (Falls doch etwas schiefgeht, ist der package-ordner wsl nicht dort, wo er hinsollte)

!!!!!

Wenn Euer Server/Client/Dienst selbst ein eigenes Package ist:

    - Hauptverzeichnis: der "VSProjekt" Haupt-Ordner
    - Euer Package befindet sich ebenfalls im selben Hauptverzeichnis wie der "storageSystem" Ordner
    - Importiert wie gehabt das "storageSystem" Package in einer eurer Klassen-Dateien:     "import storageSystem.*;"
    - Kompilieren EURES Packages geschieht dann ja aus dem Hauptverzeichnis heraus wie folgt:
        javac Client/*.java -d .
    - Das "storageSystem" Package sollte dann beim Ausführen, direkt erkannt werden, ohne dass man irgendetwas weiter angeben muss
    - Ausführen geschieht ja ebenfalls aus dem Hauptverzeichnis:
        java Client/client
